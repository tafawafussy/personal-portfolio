# personal-portfolio
A modern, responsive personal portfolio website built with React and deployed using cloud platforms as part of the **Cloud Computing** module assignment.

##  Live Links

- **Frontend (Vercel)**: [https://personal-portfolio-lake-xi.vercel.app/#projects]
- **Backend (Render)**: [https://portfolio-backend-uk94.onrender.com]

##  Features

- Fully responsive design (Mobile + Desktop)
- Dark/Light mode support
- Smooth scrolling navigation
- Contact form with backend integration
- Clean and professional UI/UX
- Deployed using industry-standard cloud platforms

##  Tech Stack

##  Frontend
- React + Vite
- Tailwind CSS
- Lucide React (Icons)

##  Backend
- Node.js
- Express.js
- CORS enabled

##  Deployment
- Frontend: Vercel
- Backend: Render
